Grid

Basic Grid
The grid container has a class of grid and each cell has the following classes: grid__cell, grid__cell--span-6@sm and grid__cell--span-3@lg

Grid with gutters
Same as the above example but with gutters. The grid container now has the following classes: grid, grid--gutters-single and grid--gutters-double@xl. You can change the gutters class to single, double or triple.

Grid with different vertical and horizontal gutters
The grid container now has the following classes: grid, grid--gutters-single, grid--gutters-double-horizontal@lg and grid--gutters-triple-vertical@lg.
