CSS Grid

Basic Grid
The grid container has a class of .o-grid and each cell has the following classes: .o-grid__cell, .o-grid__cell--span-6@sm, .o-grid__cell--span-3@lg

Basic Grid with gutters
Same as the above example with single gutters. The grid container now has the following classes: o-grid and o-grid--gutters-single. You can change the class to single, double or triple.
