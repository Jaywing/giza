Flex Grid

Basic Grid
The grid container has a class of o-flex-grid and each cell has the following classes: o-flex-grid__cell, o-flex-grid__cell--span-6@sm and o-flex-grid__cell--span-3@lg

Grid with gutters
Same as the above example but with gutters. The grid container now has the following classes: o-flex-grid, o-flex-grid--gutters-single and o-flex-grid--gutters-double@xl. You can change the gutters class to single, double or triple.

Grid with different vertical and horizontal gutters
The grid container now has the following classes: o-flex-grid, o-flex-grid--gutters-single, o-flex-grid--gutters-double-horizontal@lg and o-flex-grid--gutters-triple-vertical@lg.
